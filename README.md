## 📌 About the project:

- 📝 Task annotation system
- 📋 Task management
- 💾 Data saving using JSON
- ⚙️ Customizable and editable statuses without management

### ⚙️ Runtime used:

- 🟢 [Node.js](https://nodejs.org/)

### 📚 Libraries explored in the project:

- 🎨 [Chalk](https://npmx.dev/package/chalk)
- 💬 [@clack/prompts](https://npmx.dev/package/@clack/prompts)

# 🚀 How to use:

- ⬇️ Install Node.js from the appropriate website.
- 📥 To clone the repository, use the following command in the terminal:

    - git clone <repository_url>

- 📦 After cloning the repository, use the following commands to install the libraries:

    - npm install @clack/prompts
    - npm install chalk

- ▶️ After completing all the previous steps, start the project using:

    - node src/index.js
